# 🍱 ไดอารี่อาหารรายวัน

แอปบันทึกอาหารที่กินในแต่ละวัน แล้วสรุปออกมาเป็นรูปสไตล์หน้าไดอารี่ลายจุด ดาวน์โหลดเป็น PNG หรือแชร์ต่อได้

**เปิดใช้งาน:** https://njposatorn.github.io/food-diary/

## ความสามารถ

- บันทึก 3 มื้อต่อวัน (เช้า / กลางวัน / เย็น) เลือกวันที่ได้ แสดงเป็นภาษาไทย + พ.ศ.
- **คลังเมนู** เก็บเมนูที่กินบ่อย ไม่ต้องกรอกค่าโภชนาการใหม่ทุกครั้ง — รองรับทั้งแบบต่อ 100 กรัม และต่อหน่วย (ฟอง / ชิ้น / กล่อง)
- คำนวณแคลอรี่และโปรตีนอัตโนมัติ ทั้งรายรายการ รายมื้อ และรวมทั้งวัน
- แนบรูปอาหารได้หลายรูปต่อมื้อ บีบอัดให้อัตโนมัติก่อนเก็บ
- **สร้างรูปสรุป** เป็นหน้าไดอารี่พื้นหลังลายจุด รูปอาหารซ้าย รายการขวา (ไม่มีตัวเลขแคล/โปรตีนในรูป)
- เพิ่มลงหน้าจอโฮมได้ มีไอคอนแอปในตัว เปิดแบบเต็มจอ

## การใช้งาน

เปิดลิงก์ด้านบนได้เลย ไม่ต้องติดตั้งอะไร ใช้งานแบบออฟไลน์ได้หลังเปิดครั้งแรก

**เพิ่มลงหน้าจอมือถือ**
- iPhone: ปุ่มแชร์ → "เพิ่มไปยังหน้าจอโฮม"
- Android: เมนู ⋮ → "เพิ่มลงในหน้าจอหลัก"

## เรื่องข้อมูล

ข้อมูลทั้งหมดเก็บอยู่ใน `localStorage` ของเบราว์เซอร์ที่เปิด — ไม่ถูกส่งขึ้นเซิร์ฟเวอร์ใดทั้งสิ้น

ข้อควรระวัง:
- ข้อมูล**ไม่ sync ข้ามเครื่อง** เปิดจากมือถือกับจากคอมจะเป็นคนละชุด ควรใช้เครื่องเดียวเป็นหลัก
- ถ้าล้าง "ข้อมูลเว็บไซต์" ของเบราว์เซอร์ บันทึกจะหายไปด้วย

## โครงสร้าง

ทั้งแอปอยู่ใน `index.html` ไฟล์เดียว ไม่มี dependency ไม่ต้อง build

| ส่วน | ที่อยู่ |
|---|---|
| คำนวณโภชนาการ | `factor()` / `mealTotal()` / `dayTotal()` |
| หน้าบันทึก | `renderLog()` |
| คลังเมนู | `renderLib()` / `renderLibList()` |
| วาดรูปสรุปด้วย Canvas | `makeImage()` |
| ไอคอนแอป | `buildAppIcon()` / `installAppIcons()` |
